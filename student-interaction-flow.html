Student Interaction Flow

Usecase:

Rob is going to fullstack bootcamp and its Capstone idea/selection time

Preconditions:

  • Pick device to use the app.
  • Turn on device.
  • Navigate to app then select app.

Interaction Flow

  • User goes to login page.
  • Log page opens with prompt, to login as student/instructor or sign up.
  • User clicks sign up.
  • User is routed to a sign-up page with Email, Username, and create password.
  • User fills out required fields.
  • User is sent a conformation email and is directed to check their email.
  • After email is confirmed student is aloud into voting area.
  • Voting area has an input and add area for User idea's
  • User inputs idea
  • The idea is added to a list of ideas
  • The ideas are sorted with instructor user
  • The instructor gives instructions
  • The User follows instructors instructions
  • A new list of ideas is pushed to user for a vote
  • Voting continues based on instructors instructions
  • After voting, instructor pushes votes out for ranking
  • User rates votes from most to least favorite ideas with numerical values
